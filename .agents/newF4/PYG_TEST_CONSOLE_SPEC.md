# PYG_TEST_CONSOLE_SPEC.md
**Испытательный Полигон Пигмалиона**
Версия: 1.0 | Дата: 31 мая 2026 | Phase: 3.8 → 4.0

## Цель
Создать удобный инструмент для **ручного и полуавтоматического** прохождения полного жизненного цикла О.К. через OpenCode Terminal.

## Основные блоки

### 1. Time Machine (Управление временем)
- Переключение между реальным и тестовым временем
- Быстрые переходы: +1ч, +6ч, +12ч, +24ч, Midnight, Silence Phase, Burn Phase

### 2. O.K. Manager
- Создание О.К. с проверкой Grammar Engine
- Просмотр списка активных О.К.
- Проверка % заполнения профиля

### 3. Emission Console
- Эмиссия 3–13 У.Е. от выбранного О.К.
- Выбор триад
- Автоматическая проверка лимита

### 4. Transfer Console
- Передача У.Е. (с проверкой правил: max 5, max 1 из триады)
- Автоматическая трансформация в У.М. у получателя

### 5. Burn Simulator
- Запуск burn-cycle
- Просмотр, что сгорит, а что превратится в У.М.

### 6. Replay Console
- Запуск полного Replay
- Отчёт: mismatch, количество У.Е./У.М., ro_dag_edges

### 7. Presence & ro.DAG Viewer
- Проверка видимости О.К. в /api/presence
- Визуализация графа связей

### 8. Version Switcher
- Переключение между версиями (v0.4.1.TEST, v0.5 и т.д.)

## Критерии успеха полного цикла
- У.Е. корректно превращаются в У.М.
- У.М. переживают burn
- Replay = 0 mismatch
- Все О.К. видны в Presence
- Соблюдены фазы тишины и лимиты

4. Объявить переход в Фазу 4 (Наблюдение)
