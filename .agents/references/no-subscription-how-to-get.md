# Как получить ChatGPT Plus (или альтернативу) из России — 2026

> Reference для скилла `paperclip-deploy`. Без активной подписки (ChatGPT Plus для Codex CLI или Claude Max для Claude Code) или API-ключа скилл бесполезен — оркестратор Paperclip авторизуется через `*_local` adapter, который использует подписку провайдера, а не API-ключ.
>
> **Быстрый ответ:** за 1-2 часа — через платёжный посредник (Oplatym, GGSel). Долго и суверенно — своя зарубежная карта (KZ/AM) + виртуальный номер. **Лучшая альтернатива подписке** — OpenRouter с оплатой криптой.

## Путь 1 · Регистрация аккаунта OpenAI из РФ

### 1.1 IP — первый фильтр

OpenAI делает risk-assessment по IP **до** того как пользователь увидит форму регистрации. Типовые ошибки:
- `OpenAI's services are not available in your country` — geo-classified restricted
- `Too many signups from this network` — shared VPN exit-node, засвеченный
- Phone verification loop — средний risk-score

**Требования к IP при регистрации:**
- Статичный residential IP (не datacenter)
- Страна из whitelist: США, Германия, Нидерланды, Великобритания, Польша, Швейцария
- Не shared (бесплатные VPN в бан-листах)
- Корректная DNS-резолюция через тот же канал (без утечек)

**Что работает:** Mullvad (платный), ProtonVPN (платные тарифы), Windscribe с dedicated IP. Собственный SS-прокси на VDS в NL/DE/US.

**Что НЕ работает:** бесплатные VPN (WARP, Proton Free, OperaVPN), популярные РФ-ориентированные (Red Shield) — shared ranges в чёрном списке.

### 1.2 Номер телефона — только SMS

- Верификация **только SMS** или WhatsApp (в поддерживаемых странах)
- Email / звонок не работают
- **VoIP заблокированы** — Google Voice, Skype, TextNow
- Россия, Беларусь — в unsupported countries
- Русский номер через сторонний OTP — не работает, OpenAI блокирует префикс `+7` на форме

**Сервисы виртуальных номеров:**

| Сервис | Цена/активация | Успешность с OpenAI | Комментарий |
|---|---|---|---|
| SMS-Activate | 5–20 ₽ | ~50% (номера сожжены) | Массовый, номера быстро помечаются как VoIP |
| SMS-Act | $0.05–0.20 | 95%+ (по их данным) | Платформа под OpenAI, принимает USDT |
| Tiger-SMS | 10–30 ₽ | ~60% | РФ-ориентированный сервис |
| 5sim | 10–25 ₽ | ~55% | Массовый |
| sms24.me | 0 (публичные номера) | 1–5% | Публичные номера читают все — код не ваш |

**Рабочая тактика:**
1. Брать номер стран вне массового бана: Бразилия, Индия, Таиланд, Филиппины, Нигерия — проходят чаще, чем Европа/США
2. Один номер = один аккаунт OpenAI навсегда (номер помнится 30 дней после удаления аккаунта)
3. Если первый номер не прошёл — следующий у другого провайдера

### 1.3 Email

- Gmail — лучше всего
- ProtonMail — ок, но иногда в антифрод-списках
- Yandex / Mail.ru — не брать, часто блокируются

**Правило:** телефон при регистрации и карта при оплате — одной страны (или близкой по риск-профилю). После регистрации **не менять** страну в Settings → Region.

## Путь 2 · Зарубежная карта для оплаты $20/мес

OpenAI использует Stripe, который фильтрует карты по BIN. Российские BIN (Сбер, Тинькофф, ВТБ, Альфа) — в автоматическом reject-листе, **включая карты с логотипами Visa/Mastercard**.

### 2.1 Физические карты (стран ЕАЭС и ближнего зарубежья)

| Карта | Страна | Как получить | Стоимость | Работа с OpenAI | Риск |
|---|---|---|---|---|---|
| **Freedom Finance (Freedom Card)** | KZ | Удалённо через посредников (EasyCrds), нужен загранпаспорт, регистрация во Freedom Broker ($125 депозит, $100 возвращается) | ~34 900 ₽ за услугу + $125 | Работает (подтверждено гайдами 2026) | Низкий, но окно закрывается — KZ обсуждает ограничения на ИИН нерезидентам |
| **Forte Bank** | KZ | Только с физическим визитом | Стоимость поездки | Работает | Низкий |
| **Kaspi Gold** | KZ | Только лично, требует ИИН и местный адрес | Только поездка | Работает | Низкий |
| **AmeriaBank / Inecobank** | AM | Лично в Ереване, 1–2 дня | 500–2000 ₽ | Работает | Низкий |
| **Ziraat / Denizbank** | TR | Лично в Стамбуле, нужна виза/ВНЖ + ИНН Турции | $50–100 | Работает | Средний — турецкие банки стали отказывать гражданам РФ с 2024 |
| **Компаньон Банк / Бакай Банк** | KG | Удалённо через посредников | 15–30 тыс. ₽ | Работает | Средний |

**Freedom Finance KZ** — исторически главный выбор. Апрель 2026: банк поддерживает eResidency с удалённым ИИН и счётом. Но постановление АРРФР №18 позволяет отказ по AML-риску, банк фильтрует нерезидентов РФ. Посредники EasyCrds/VisaMasterCardRussia работают, но канал может закрыться в любой момент.

### 2.2 Виртуальные карты (РФ-финтехи под подписки)

Схема: пополнение рублями через СБП → конвертация → траты по Visa/MC с BIN-ом ЕС/США/ЕАЭС.

| Сервис | BIN страна | Выпуск | Обслуживание | Пополнение | Работа с OpenAI |
|---|---|---|---|---|---|
| **Wanttopay Easy** | US/ЕС | $10 | $6/мес (Plus) или бесплатно | Рубли (СБП), крипта | **Работает** |
| Wanttopay Prepaid | US/ЕС | $14 | Нет | Как Easy | Работает |
| Wanttopay Smart | US/ЕС | $15 | $6/мес | Как Easy (+ Apple Pay) | Работает |
| **Плати по миру** | US | 0 ₽ (по акции) | Без абонплаты | СБП, крипта | **Работает** |
| **e.PN** | US/ЕС/Чили/Таиланд/Индия/UK и др. | $2–5 | $2+/мес | Крипта, Capitalist, Payeer, PayPal, Alipay | Работает |
| FlexCard | ЕС | от 299 ₽ | По тарифу | Рубли | Работает |
| Xcards | 43 разных BIN | $1–3 | Зависит | Рубли, крипта | Работает частично |
| **Zarub** | Зарубежный | ~500 ₽ | Без абонплаты | Рубли (СБП), крипта | **Работает для recurring** |

**Нюансы:**
- Первый платёж может `decline` — нужен запас $5-10 сверх подписки для compliance-чека Stripe
- Не все BIN одинаково «пропускаемы» OpenAI — e.PN заявляют 43 BIN, но рабочих для OpenAI 3-5
- Рублёвое пополнение идёт по внутреннему курсу +3-5%
- Проверять что карта поддерживает recurring (не все)

### 2.3 Крипто-карты

Большинство закрыто для РФ в 2026:
- **Binance Card** — недоступна россиянам с 2022
- **OKX Card** — KYC для россиян закрыт
- **Bybit Card** — закрыт
- **Pyypl** (ОАЭ) — через посредников ~30-50 тыс. ₽ за оформление

Остались виртуалки (e.PN), где крипта — способ пополнения, не эмитент.

### 2.4 Freedom24 (брокер, не карта)

Freedom24 / Freedom Finance Europe — кипрский брокер, у россиян был популярен для SWIFT-реквизитов. **В 2026 актуальность упала**: 12-й пакет санкций ЕС осложнил платежи от резидентов РФ. Для ChatGPT — не оптимально.

## Путь 3 · Платёжные посредники (самый быстрый путь)

Нет карты и нет времени — оптимум. Схема: платите рублями через СБП, сервис покупает подписку с их зарубежной карты.

| Сервис | Способ | Время | Комиссия на $20 | Возврат |
|---|---|---|---|---|
| **Oplatym.ru** | Предоплаченная карта (аккаунт ваш) или передача доступов | 15–60 мин | 550–650 ₽ (итого 2400–2850 ₽) | Да |
| **GGSel** (маркетплейс) | Готовый аккаунт или активация на ваш | 10–30 мин | ~2200–2900 ₽ | Зависит от продавца |
| **Aifory.pro** | Аналогично Oplatym | 15–60 мин | ~550–800 ₽ сверху | Да |
| **Плати по миру** | Выпускают виртуалку на ваш аккаунт | 1–2 часа | Разово за карту + per-transaction | Возврат по карте |

**Два режима работы:**

1. **Предоплаченная карта** (рекомендуется): выпускают виртуалку на ваше имя, вы сами привязываете в ChatGPT. Аккаунт 100% ваш. Минус — разовая плата за карту.

2. **Передача доступов**: даёте им логин/пароль ChatGPT, они логинятся и привязывают карту. Быстрее, но:
   - У них остаются credentials
   - Привязка с IP другой страны = red flag → возможный бан

**Правило:** выбирайте предоплаченную карту, не отдавайте логины.

## Путь 4 · Альтернативы подписке — РЕКОМЕНДУЕТСЯ

### 4.1 OpenRouter + USDC — лучшая альтернатива для Paperclip

**Что это:** агрегатор 400+ LLM (Claude, GPT, Gemini, DeepSeek, Llama) с единым OpenAI-совместимым API. Pay-as-you-go, без подписки.

**Почему идеально для Paperclip:** adapter `openclaw_gateway` из коробки работает через OpenRouter. Для этого сценария подписка не нужна вообще.

**Оплата из РФ:**
- **USDC через Ethereum / Polygon / Base** — прямой канал, комиссия 5%. Без VPN, без карты
- Через посредников (Oplatym, aitunnel.ru) — рублями, 5-10% сверху

**Цены на 1M токенов (апр 2026):**

| Модель | Input / Output |
|---|---|
| Claude Sonnet 4.5 | $3 / $15 |
| GPT-4.1 mini | $0.40 / $1.60 |
| DeepSeek V3 | $0.30 / $1.20 |

**Для типового CEO-агента в Paperclip** (~100K токенов/день): $10-30 в месяц — **дешевле Plus-подписки**.

**AITunnel.ru** — российский прокси к OpenRouter, пополнение рублями, работает без VPN. Формат моделей тот же (`openai/gpt-5`, `anthropic/claude-sonnet-4`). Наценка — 5-10%.

### 4.2 API OpenAI вместо подписки

- Pay-as-you-go, типично $5-20/мес для умеренного использования
- Нужен аккаунт OpenAI + platform.openai.com + пополнение — те же проблемы с картой что и у Plus
- **НЕ подходит для Paperclip adapter `codex_local`** — он использует `chatgpt.com/backend-api/codex` через OAuth подписки, не API-ключ

### 4.3 Claude API / Claude Max

- Тот же путь: не-РФ карта + VPN + номер
- **Но:** Anthropic жёстче банит подписки за использование в оркестраторах. Paperclip-скилл по умолчанию рекомендует Codex CLI + ChatGPT Plus — безопаснее
- Claude Max — $100/мес (Max 5x) или $200/мес (Max 20x). Окупается только при интенсивном использовании

### 4.4 Gemini — бесплатный tier

- **aistudio.google.com** — доступен с РФ IP, без VPN, без карты
- Бесплатный Gemini 2.5 Flash — 15 req/мин, миллионный контекст
- Подходит для **части** сценариев Paperclip (но не для Codex CLI который привязан к OpenAI endpoint'ам)

### 4.5 Серый рынок — аренда / покупка аккаунтов

**Не рекомендуется для продакшена.** Источники (Avito, Kwork, закрытые TG-каналы) продают готовые аккаунты с подпиской за 800-1500 ₽/мес. Дешевле официальной цены, но:
- Аккаунт могут забанить в любой момент (один аккаунт = много пользователей, Stripe видит multi-IP)
- Внутри — чужая история
- При бане все настройки теряются
- В Paperclip-контексте: бан в середине workflow → оркестратор падает

**Для бизнес-задач (Paperclip 24/7) — только свой аккаунт.**

## Путь 5 · Оплата когда всё готово

1. Войти в ChatGPT через **тот же** VPN-узел, на котором регистрировался (одна страна!)
2. Settings → My Plan → Upgrade to Plus
3. Ввести карту с зарубежным BIN
4. При `Your card was declined`:
   - Проверить сумму на карте ≥ $25 (Stripe делает pre-auth)
   - Проверить включён ли 3D-Secure (для большинства виртуалок — да)
   - Попробовать другой VPN-узел той же страны
   - Подождать 30-60 минут (временные фильтры Stripe)
   - **Не перебирать карты >3 раз подряд** — аккаунт в риск-ревью

### Origin mismatch

Если IP — Нидерланды, номер — Таиланд, карта — Казахстан → Stripe видит три разные страны → флаг.

**Идеал:** одна страна для IP + номера + карты.
**Минимум:** одна страна для IP и номера, карта может быть другой.

## Путь 6 · Безопасность аккаунта после оформления

Правила долгой жизни:

1. **Не менять страну привязки** — после регистрации `Settings → Region` не трогать
2. **Стабильный VPN-узел** — Mullvad/ProtonVPN с dedicated IP ($5-10/мес сверху). Shared exit-nodes ротируются, OpenAI видит разные IP
3. **Никогда не логиниться с РФ IP** — одна случайная сессия = red flag навсегда
4. **Одно устройство = один профиль** — один и тот же Chrome-профиль ходит в один аккаунт. Не 5 разных устройств
5. **Не перебирать карты при decline** — один-два раза ок, пять — watch-list
6. **Не запрашивать refund** — chargeback = моментальный бан на всей семье аккаунтов (включая будущие регистрации с того же номера)

**Что делать если забанили:**
- Обращение в help.openai.com → обычно формальные отказы
- Шанс восстановления: ~10% при неочевидной причине (IP), ~0% при chargeback или нарушении ToS
- Номер после бана в 3 аккаунтах блокируется навсегда

## Итог для Paperclip-сценария

### Быстрый путь (сегодня, 1-2 часа)
1. Купить подписку ChatGPT Plus через **Oplatym.ru** с «предоплаченной картой» — 2400-2850 ₽
2. Подписка на свой аккаунт ChatGPT, регистрация с VPN (NL/DE) + виртуальный номер через SMS-Activate
3. В Paperclip — `codex login --device-auth` под этой подпиской, токен в `/opt/paperclip/.codex/auth.json`

### Долгосрочный суверенный путь (2-7 дней)
1. Freedom Finance KZ через EasyCrds (~35 000 ₽ единоразово + $125 брокер-депозит, $100 возвращаются)
2. Виртуальный номер Таиланда/Индии — 15-30 ₽
3. Mullvad/ProtonVPN с dedicated IP в NL/DE — $5-10/мес
4. Оплата Plus напрямую со своей карты — $20/мес чистыми

### Дешёвый обход без подписки — рекомендация
1. **OpenRouter + USDC через Polygon** (5% комиссия) — $10-30/мес для типовых задач
2. В Paperclip — adapter `openclaw_gateway` вместо `codex_local`
3. Не нужен ни ChatGPT Plus, ни Claude Max, ни VPN

### Что НЕ делать
- Покупать аккаунт на Avito/Kwork для продакшн-Paperclip — бан в любой момент
- Использовать бесплатные VPN (WARP, Opera) — IP в бан-листах
- Регистрироваться с русским номером через OTP — блокировка на уровне `+7`
- Запрашивать chargeback — навсегда теряете возможность регистрации с этого номера/карты
